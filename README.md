**PUNTO TRES**

**Hardware**:*El hardware es la parte física de una computadora o dispositivo electrónico, es decir, todo lo que puedes ver y tocar, ejemplo:el procesador, la memoria, el disco duro, la pantalla, el teclado*.
**Software**:*es la parte lógica o intangible de una computadora, es decir, los programas, aplicaciones y sistemas que le dicen al hardware qué hacer, ejemplo: el software es el cerebro que le da órdenes*.
**Los Lenguajes de Programación**:*es un conjunto de instrucciones y reglas que las personas usamos para decirle a una computadora qué hacer, los lenguajes de programacion usan palabras, simbolos y reglas que los programadores entienden con facilidad*

**Python**:*Fácil de entender, ejemplo:print("Hola, mundo")
Esto le dice a la computadora que muestre el texto “Hola, mundo” en la pantalla*.

**Variables, valores y oeperadores**:
**Variable**:*es como una caja donde guardas un dato (un número, texto, etc.) para usarlo después*.
**Valores**:*son los datos reales que guardan las variables*.
**Operadores**:*sirven para realizar acciones o comparaciones con las variables y valores(suma,resta,multiplicación y división )*.

**Orden de las operaciones en Python**:*Python sigue un orden parecido al de las matemáticas:Paréntesis → Potencias → Multiplicación/División → Suma/Resta*

**operaciones en cadena de texto**:*Una cadena de texto (o string) es una secuencia de caracteres;Python la trata como un dato que puedes juntar, repetir o analizar,ejemplo:mensaje = "Hola, Yurani"*

**Lectura y escritura de valores y variables**: *Para que el usuario escriba algo, se usa la función input().
Todo lo que se escribe con input() se guarda en una variable y siempre se lee como texto (string)*.

**Alcance de las variables globales y locales**

**Globales**:*se declara fuera de las funciones, y se puede usar en todo el programa*.

**Locales**:*se crea dentro de una función y solo existe dentro de ella*.

**Condicionales**:
*El tema de los condicionales es muy importante en programación, porque permiten que el programa tome decisiones dependiendo de una condición (algo que puede ser verdadero o falso)*.

**CONCLUSION**:

*Aprendí que el hardware, el software y los lenguajes de programación trabajan juntos para que la computadora funcione. Con Python podemos dar instrucciones, usar variables y condicionales para que el programa tome decisiones y realice diferentes tareas*.
